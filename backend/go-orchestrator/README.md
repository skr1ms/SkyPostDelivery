# Go Orchestrator

## Overview (EN)
- Language: Go 1.21+ (Gin, gRPC, SQLC/pgx).
- Role: central API for authentication, orders, deliveries, parcel automats, drones, goods.
- Integrations: RabbitMQ (delivery tasks), PostgreSQL, MinIO, drone-service gRPC, SMSAero, Firebase push.
- Metrics: Prometheus endpoint on `:9091/metrics`.

### Useful Commands
- `make run` — start HTTP (8080) and gRPC (50052) servers locally.
- `make test` or `go test ./...` — run unit tests.
- `sqlc generate` — regenerate repository code from SQL queries.

### Key Directories
- `cmd/app/` — application entry point.
- `internal/controller/http` — REST handlers.
- `internal/controller/grpc` — gRPC services.
- `internal/usecase` — business logic.
- `internal/usecase/repo` — PostgreSQL repositories (generated by SQLC).
- `pkg/pb` — generated protobuf clients.
